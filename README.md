# NoReference-Image-Quality-Assesment

To install, download OpenCV - v.4.10 (https://github.com/opencv/opencv/releases)
Nevegate to directory:
    run $mkdir build
    run $cmake ..
    run $make 
    run $sudo make install

To run application, nevigate to root directory:
    run $mkdir build
    run $cmake ..
    run $make
